# premex-ab/homebrew-tap

Homebrew tap for [Premex AB](https://github.com/premex-ab) CLI tools.

## Install

```bash
brew tap premex-ab/tap
```

## Available formulas

| Formula | Source repo |
|---|---|
| `adb-connect` | [premex-ab/adb-connect](https://github.com/premex-ab/adb-connect) |

## How this tap is maintained

Formulas under `Formula/` are written by [GoReleaser](https://goreleaser.com/)
from each source repo's release workflow, authenticated via the Premex Bot
GitHub App. Do not edit `Formula/*.rb` files by hand — they will be overwritten
on the next upstream release.
